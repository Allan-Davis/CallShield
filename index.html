Install Guide

CallShield isn't on the Play Store — here's exactly what to do.

1
Download the APK Tap the Download button. The file saves to your Downloads folder automatically.
2
Allow Unknown Sources Settings → Apps → Your browser (or Files app) → Install unknown apps → Allow.
3
Open & Install the APK Open Files app → Downloads → tap CallShield APK → Install.
4
Grant Permissions Open CallShield → tap "Grant Permissions & Continue" → accept all prompts. Required for blocking to work.
