Installation

Install guide

CallShield isn't on the Play Store — here's exactly what to do.

01
Download the APK

Tap the Download button on this page. The file saves to your Downloads folder automatically.

02
Allow Unknown Sources

Go to Settings → Apps → your browser (or Files app) → Install unknown apps → Allow.

03
Open & Install the APK

Open Files app → Downloads → tap the CallShield APK → Install.

04
Grant Permissions

Open CallShield → tap "Grant Permissions & Continue" → accept all prompts. These are required for blocking to work.

⚠️
See a Play Protect warning? Tap More details → Install anyway. Android shows this for any app outside the Play Store. CallShield is completely safe — your data never leaves your device.
